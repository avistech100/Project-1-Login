# Project: Login Page

## Description
This project implements a simple login page with two login options:
1. Email
2. Password

## Features
- User authentication through email and password.
- Secure password handling.
- Intuitive user interface for seamless login experience.

## Technologies Used
- HTML
- CSS

## Setup Instructions
1. Clone the repository.
2. Open the project in your preferred code editor.
3. Run the project locally using a web server or by opening the HTML file in a web browser.

## Usage
1. Enter your email and password in the respective fields.
2. Click on the login button to access the system.
